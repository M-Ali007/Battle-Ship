# Battleship

A single-player Battleship game built with Python and Pygame. Three ships are hidden on a 7x7 grid — click cells to find them all.

## Gameplay

- Click any cell on the blue grid to fire a shot
- **Red circle** = hit, **white ring** = miss
- Find all 9 ship cells (3 ships x 3 cells) to win
- A green sweep animation plays on victory
- Press **R** at any time to restart with new ship positions
- Your move count is tracked in the top-right corner

## Requirements

- Python 3.12+
- Pygame 2.6.1+

## Setup & Run

```bash
pip install pygame
python "Ali_Mustafa - Battleship.py"
```
